This project is a web-based phishing email detection system built with Flask. It allows users to:

  • Submit suspicious email content for analysis
  • Detect phishing using simple rules + optional AI (Mistral via Ollama)
  • Check sender domains against VirusTotal
  • View past analysis history with timestamps
  • Register/login securely
Technologies Used:
  • Python (Flask, sqlite3, re, requests)
  • Flask-Login for authentication
  • Bootstrap for UI
  • Ollama & Mistral (optional AI explanations)
  • VirusTotal API for domain safety
How It Works:
  1. User logs in or registers
  2. Submits suspicious email content
  3. System scores the content + calls AI (optional)
  4. Results are displayed and stored in user history
